## Структура

| Путь | Формат | Модуль |
|------|--------|--------|
| `samples/payloads.csv` | CSV | WAF payload |
| `samples/sample_network_flows.csv` | CSV | Network anomaly |
| `raw/` | — | Внешние датасеты (не в git) |

---

## Образцы payload

Колонки типичного CSV:

| Колонка | Описание |
|---------|----------|
| `payload` | Строка запроса / тела |
| `label` | `benign` / `malicious` (или бинарный класс) |

Используются в `waf_payload.train` и unit-тестах.

---

## Сетевые потоки

Признаки потока (пример): `duration`, `src_bytes`, `dst_bytes`, `protocol`, `service` и др. — см. конфиг `configs/network_anomaly.yaml`.

---

## Правила использования

| Правило | Требование |
|---------|------------|
| Права | Обрабатывайте только данные, на которые у вас есть право |
| Payload | Не направляйте образцы против чужих production-систем |
| Raw | Крупные датасеты кладите в `raw/` локально; не коммитьте секреты |
