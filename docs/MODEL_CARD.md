## Назначение

| Модуль | Задача |
|--------|--------|
| **WAF payload** | Классификация категорий вредоносных веб-payload (защитная разметка) |
| **Network anomaly** | Метки / скоры аномальности сетевых потоков на демо-признаках |

Модели **базовые** (baseline): подходят для обучения, портфолио и локальных экспериментов.

---

## Допустимое применение

- Обучение и демонстрация ML-пайплайнов.
- Триаж данных, на которые у вас есть **явное право** обработки.
- Собеседования и курсовые на синтетических / разрешённых выборках.

---

## Недопустимое применение

| Запрет | Пояснение |
|--------|-----------|
| Атаки на чужие системы | Только defensive use |
| Автоблок без человека | Не единственный источник решений в production |
| Заявления о «production accuracy» | Без валидации на ваших данных |

## Метрики и честность

Метрики хранятся в joblib (`metrics`) после `train` / `evaluate`.

| Ситуация | Поведение API |
|----------|----------------|
| Свежий клон без обучения | Сводка может показывать нули — намеренно |
| После обучения | Честные цифры из артефакта |

## Ограничения

| Фактор | Риск |
|--------|------|
| Малые demo-датасеты | Нестабильная точность, плохая переносимость |
| Адаптация атакующих | Статические модели дрейфуют |
| Rule-based слой | Снижает неоднозначность, не заменяет полноценный WAF |

Мониторинг дрейфа: [`DRIFT.md`](DRIFT.md).
