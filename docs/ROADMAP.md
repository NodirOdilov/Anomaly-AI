# Дорожная карта

- **v1.0 MVP**: полноценное демо FastAPI + React + Docker + тесты + честная отчётность по метрикам.
- **v1.1 Лучшие датасеты**: интеграция крупных авторизованных корпусов; улучшение отчётов по оценке.
- **v1.2 Аутентификация**: API-ключи или OAuth для многопользовательских демо.
- **v1.3 Интеграция с SIEM**: экспорт алертов в CEF/JSON на webhook или syslog.
- **v1.4 Nginx / WAF**: паттерн sidecar-сервиса скоринга (по-прежнему только защита).
- **v2.0 Production-мониторинг**: детекция дрейфа, дашборды производительности, конвейер переобучения.

Подробный план v2.x: [`MEGA_PLAN.md`](MEGA_PLAN.md).
