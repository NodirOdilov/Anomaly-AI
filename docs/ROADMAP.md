# Roadmap

- **v1.0 MVP**: Full-stack demo with FastAPI + React + Docker + tests + honest metrics plumbing.
- **v1.1 Better datasets**: Integrate larger authorized corpora; improve evaluation reports.
- **v1.2 Authentication**: API keys or OAuth for multi-user demos.
- **v1.3 SIEM integration**: Export alerts as CEF/JSON to a webhook or syslog sink.
- **v1.4 Nginx / WAF integration**: Sidecar scoring service pattern (still defensive).
- **v2.0 Production monitoring**: Drift detection, performance dashboards, and retraining pipeline.
